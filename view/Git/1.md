# Git 概念

1. 版本库👉.git文件夹
    - 当我们使用git管理文件时，比如git init时，这个时候，会多一个.git文件，我们把这个文件称之为版本库。
    - .git文件另外一个作用就是它在创建的时候，会自动创建master分支，并且将HEAD指针指向master分支。
2. 工作区
    - 本地项目存放文件的位置
3. 暂存区 (Index/Stage)
    - 顾名思义就是暂时存放文件的地方，通过是通过add命令将工作区的文件添加到缓冲区
4. 本地仓库（Repository）
    - 通常情况下，我们使用commit命令可以将暂存区的文件添加到本地仓库
5. 远程仓库（Remote）
    - 服务器上的git仓库
